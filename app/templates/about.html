About Eklogi

A site to explore the U.S. political environment. Eklogi tracks politicians of the current Congress and shows how they vote, which committeess they belong to, and the progression of bills.

Data + Tools

We scraped Propublica API to get the necessary info for members, committees, bills, and votes.

Hosting: We use AWS to host our web application

API Documentation: We use Apiary to document the Probublica API

GUI: Utilize Bootstrap for dynamic rending of our content.

Frontend: We implement our pages with Javascript, particularly React

Backend: Our templates are served using Flask

Database: We have setup our models for use with SQLAlchemy to achieve persistence

Domain: Our url was acquired through Namecheap

Logistics: Github for version control and project management

Communication: Slack with integrations to keep the team up-to-date
